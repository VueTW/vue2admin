#### Download:

Download from Github

#### Using The Command Line:

```
git clone https://github.com/harryho/Vue2Admin.git
```


#### Build Setup

``` bash
# install dependencies
sudo npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
